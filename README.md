## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites
* Text Editor
* NPM
* React Router
* React Micromodal
* Dayjs
* Normalize
* Typescript

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:MIBFB-COLLAB/MESSAGE-IN-A-BOTTLE.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
